# graph_theory_project
graph theory project during my S5 at ISEN

# Réponses aux questions :
    ** 1 -  Proposez une implémentation d’un graphe, qui représente une grille hexagonale et qui possède toutes les propriétés d’un graphe.**

    On va se tourner vers une liste de sucesseurs car un hexagone contient 6 côtés il a donc 6 voisins (sucesseurs).
    Une matrice d'adjacence aurait |S^2| en taille ce qui peut faire beaucoup pour notre graphe.
